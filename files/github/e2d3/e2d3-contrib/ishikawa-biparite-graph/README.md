ishikawa-bipartite-graph(2部グラフ)

■使い方  
1)必要項目を、下記に従いセルに作成する  
　　・1行目にタイトル文字列を記載  
　　・2行目に、カラム名称を記載  
　　・3行目以降に、描画したいデータを記載  
  
2)上記作成データの範囲を選択した状態で、E2D3のLink dataボタンを押下する。
